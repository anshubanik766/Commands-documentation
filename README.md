# Commands-documentation
commands list of slate-bot written on JS which is a multipurpose discord bot

### Made by - Vortex Labs

---

## Commands List
| Command | Description | Category |
| :--- | :--- | :--- |
| `admin` | Manages trusted users allowed to modify antinuke parameters and settings. | Antinuke |
| `antinuke` | Main configuration panel for the server antinuke security system. | Antinuke |
| `channel` | Configures limits on rapid channel creations and deletions. | Antinuke |
| `antiraid` | Toggles the server antiraid protection layer to mitigate malicious join waves | Antiraid |
| `antiraid-avatar` | Configures profile picture verification parameters for incoming accounts during raid states | Antiraid |
| `antiraid-config` | Displays the complete configuration profile and active filter states for the anti-raid system | Antiraid |
| `antiraid-massjoin` | Configures join velocity thresholds to auto-detect and mitigate mass join/bot attacks | Antiraid |
| `antiraid-newaccounts` | Configures account age verification thresholds to block or flag freshly created burner profiles | Antiraid |
| `antiraid-state` | Provides a condensed, high-level status summary of the anti-raid perimeter | Antinuke |
| `antiraid-whitelist` | Manages the exceptional user exclusion list for bypassing anti-raid perimeter triggers | Antiraid |
| `antiraid-whitelist-view` | Displays a comprehensive readout of all user IDs permitted to bypass the active anti-raid firewall gates | Antiraid |
| `automod` | Complete management dashboard for native Discord rules and AI text classification filtering. | Automod |
